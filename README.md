# DomainRegistry - Digital Domain Name Ownership Registry

A blockchain-based digital domain name ownership and transfer registry built on Stacks, providing transparent and immutable records of domain registrations and ownership transfers.

## Overview

DomainRegistry enables domain registrants to register their digital domain names with detailed purpose information while maintaining secure ownership records and facilitating domain transfers.

## Features

- Domain name registration with extension and purpose tracking
- Domain description and registration duration management
- Registration status tracking and ownership verification
- Domain ownership transfer functionality
- Comprehensive input validation and security measures

## Smart Contract Functions

### Public Functions
- `register-domain-name`: Register domain names with detailed information
- `transfer-domain-ownership`: Transfer domain ownership to new registrants

### Read-Only Functions
- `get-domain`: Retrieve domain registration information
- `get-domain-registrant`: Get domain registrant information

## Usage

Register domain names with detailed purpose and description information, track registration status and duration, and transfer ownership when needed. Build a comprehensive domain portfolio.

## Security

- Registrant verification for domain ownership transfers
- Comprehensive input validation for all domain parameters
- Status-based transfer restrictions to ensure proper workflow
- Domain name and description length validation for data integrity
